# GENERATION SETTINGS RECOMMENDATIONS

**For Midjourney:**
```
--v 6 (latest model)
--quality 2 (highest quality)
--ar 16:9 (horizontal video frame)
--seed [random] (for variety, or set number for consistency)
--style raw (for more authentic Lego render look)
--no photography (avoid photographic style)
```

**For DALL-E 4:**
```
Style: 3D render, Lego animation style, stop-motion aesthetic
Quality: High definition, 4K, detailed
Aspect: 16:9 (landscape) or 9:16 (portrait mobile)
Lighting: Professional, cinematic, clear shadows
Colors: Bright primary colors, high saturation, Lego authentic
Texture: Visible Lego brick plastic finish
```

**For Stable Diffusion XL:**
```
Positive prompts: Add "lego, 3D render, cinematic, 4K, detailed, professional"
Negative prompts: "photography, realistic humans, blurry, low quality"
CFG Scale: 7-9 (for consistency with prompt)
Steps: 30+ (for high quality)
Sampler: DPM++ or Euler (for best quality)
```
